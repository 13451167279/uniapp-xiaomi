# shop_manage

## Project setup
```
npm install
```
### 项目说明

####主页
	点击右上角图标直接去往登录
	
#### 分类
	点击商品去往详情页
		推荐、小米手机、redmi手机、游戏手机有数据
	
####详情页
	除了客服、立即购买都以实现功能
	
####购物车
	点击左上角去往分类页
	
####个人中心

####登录
	点击微信图标授权登录
	
	

### 启动服务器
```
npm start
```

运行到微信开发者工具
